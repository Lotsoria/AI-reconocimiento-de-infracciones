# Traffic Violations AI (MVP)

Detecta 4 infracciones en video:
- Paso con semáforo en rojo
- Sin casco en motocicleta
- Exceso de velocidad (estimación)
- Invasión de carril / línea continua

## Requisitos
- Python 3.10+
- GPU opcional (mejor FPS)

## Instalación
```bash
bash scripts/setup_env.sh
python scripts/download_models.py
